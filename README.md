# forexmm-cli
> Get Myanmar Exchange Rate in terminal

[![npm][npm-download]][npm-dl-url]
[![NSP Status][nsp]][nsp-url]
[![contributions welcome][contri]][contri-url]

## Installation
```
npm install -g forexmm-cli
```

## Get Exchange Rate
```
forexmm
```

## Currency Converter
```
forexmm number
```
### Example
```
forexmm 1000
```
